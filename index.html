Présentation du DOM

Le Document Object Model, in english please :), représente le 'HTML' d'une page web.
Avec JS nous allons pouvoir utiliser un ensembles d'interfaces qui travaillent ensembles pour que l'on puisse manipuler les différents éléments de notre page web
Le DOM peut-être représenté sous forme d'arbre.
Exemple: le fichier actuel, le point de départ est la balise html qui a 2 branches une repésente la balise head et la 2onde le body
ensuite head a 4 branches et body 5 branches etc...
On parle aussi de noeud le html étant le noeud racine chaque balise html étant un noeud
Tous cela paraît simple mais il y a une vraie complexité derrière ces interfaces et cela nous permet d'avoir une grande précision !

je ne peut pas toutes vous les lister il y en a beaucoup je vous présente les plus utilisées

getElementById(), getElementByClasssName(), getElementByTagName(),... comme vous pouvez le constater le nommage respecte une convention
Cette convention de nommage implique que la fonction commence par une minuscule sans espace et chaque mot suivant commence par une majuscule

il existe aussi querySelector() il permet d'affiner notre selection d'élément
